# 🎭 GS Arts Center 3D 좌석 시야 뷰어 (v3)

본 프로젝트는 Shapespark 3D 가상 시야 기술과 2D SVG 정밀 좌석 배치도를 유기적으로 연동한 **프리미엄 공연장 좌석 시야 시뮬레이션 시스템**입니다. 관객이 티켓을 예매하기 전에 가상 공간에서 좌석별 무대 시야를 미리 체험할 수 있도록 고도화된 UI/UX 환경을 제공합니다.

---

## 🌟 주요 기능 및 특장점

1. **2D-3D 유기적 연동 (Seat View)**
   - 정밀하게 매핑된 2D SVG 좌석 배치도를 통해 좌석 클릭 시 즉각적으로 Shapespark 3D 엔진의 카메라 뷰와 연동되어 실제 시야를 보여줍니다.
   - 1층, 2층, 3층의 각 블록(A, B, C) 및 실제 예매 가능한 좌석 정보를 완벽하게 지원합니다.

2. **지능형 좌석 네비게이션 (Seat Navigation)**
   - 하단 제어 바의 `이전` / `다음` 버튼 클릭 시, 데이터상 존재하지 않거나 시야가 확보되지 않은 더미 좌석("none" 타입)을 자동으로 건너뛰고 **실제 존재하는 다음 좌석으로 즉시 이동**합니다.
   - 카메라 뷰 전환 시 발생할 수 있는 중복 클릭 및 렌더링 튐 현상을 방지하기 위해 **1초 간의 입력 제한(isNavigating Lock)** 로직이 내장되어 있습니다.

3. **고품격 인터렉션 및 마이크로 애니메이션**
   - **글래스모피즘(Glassmorphism) 테마**: 극장의 품격에 맞는 고급 반투명 블랙/화이트 블러 레이아웃을 전반에 걸쳐 적용했습니다.
   - **마이크로 인터렉션**: 층 선택 버튼, 구역 선택 버튼, 기능 카드 버튼 등에 마우스 호버 시 부드럽게 위로 플로팅(`translateY(-3px ~ -6px)`)되는 큐빅 베지어 애니메이션을 탑재하여 시각적 만족도를 높였습니다.

4. **사용자 친화적 가이드 및 팝업 시스템**
   - **조작 방법 안내 (Manual Overlay)**: 깔끔한 풀스크린 반투명 레이아웃과 감각적인 닫기 버튼을 통해 사용자가 조작법을 한눈에 파악할 수 있도록 도우며, 배경의 가독성을 보장합니다.
   - **와이드 좌석 팝업**: 좌석 배치도를 넓은 시야에서 조작할 수 있도록 팝업의 크기를 화면 크기 대비 가로 90%, 세로 88%로 시원하게 확장하였습니다.

## 📁 파일 구성 설명

- `index.html`: 메인 애플리케이션 및 다양한 서비스 선택/가이드 화면 구조 정의.
- `gs_arts_center_ui.js`: 좌석 간 이동 로직, 오버레이 관리, 클릭 이벤트 등 전반적인 UI 상호작용 제어 스크립트.
- `gs_arts_center_ui.css`: 프리미엄 디자인 토큰, 모던 타이포그래피, 반응형 오버라이드 및 고품격 트랜지션 애니메이션 정의.
- `gs_arts_center_seatmap.js`: 전체 층/구역/행렬 정보 및 각 좌석별 3D 카메라 뷰(View_Name) 매핑 데이터베이스.

---

## 🚀 시작하기

1. 본 저장소를 로컬 서버 환경에 구성합니다.
2. 터미널에서 로컬 서버 배치 파일(`.\run_server.bat`)을 실행하거나 라이브 서버를 구동합니다.
3. 브라우저로 접속하여 시각화된 프리미엄 가상 투어를 경험하세요!
